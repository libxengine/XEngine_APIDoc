函数

HttpProtocol_Server2_PKTGoawayEx

功能

打包一个GOAWAY协议包

语法

BOOL HttpProtocol_Server2_PKTGoawayEx(XHANDLE xhToken, CHAR* ptszMsgBuffer, int* pInt_MsgLen, int nStreamID, int nErrorCode = 0)

参数

 参数.一：ptszMsgBuffer
  In/Out：Out
  类型：字符指针
  可空：N
  意思：输出打好包的缓冲区
 参数.二：pInt_MsgLen
  In/Out：Out
  类型：整数型指针
  可空：N
  意思：输出缓冲区大小
 参数.三：nStreamID
  In/Out：In
  类型：整数型
  可空：N
  意思：输入要操作的流ID
 参数.四：nErrorCode
  In/Out：In
  类型：整数型
  可空：Y
  意思：错误码,可以为0

返回值

类型：逻辑型
意思：是否成功
