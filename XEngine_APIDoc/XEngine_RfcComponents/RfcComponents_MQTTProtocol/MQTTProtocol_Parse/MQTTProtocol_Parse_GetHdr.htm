函数

MQTTProtocol_Parse_GetHdr

功能

获取协议头是否完整

语法

BOOL MQTTProtocol_Parse_GetHdr(LPCSTR lpszMsgBuffer, int nMsgLen, int* pInt_DTLen = NULL, int* pInt_HdrSize = NULL)

参数

 参数.一：lpszMsgBuffer
  In/Out：In
  类型：常量字符指针
  可空：N
  意思：输入要解析的缓冲区
 参数.二：nMsgLen
  In/Out：In
  类型：整数型
  可空：N
  意思：输入缓冲区大小
 参数.三：pInt_DTLen
  In/Out：Out
  类型：整数型指针
  可空：Y
  意思：输出后续大小
 参数.四：pInt_HdrSize
  In/Out：Out
  类型：整数型指针
  可空：Y
  意思：输入协议头大小(2字节头+变长负载长度)

返回值

类型：逻辑型
意思：是否成功
