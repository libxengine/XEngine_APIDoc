函数

RfcComponents_SIPClient_Create

功能

创建一个客户端

语法

BOOL RfcComponents_SIPClient_Create(XNETHANDLE *pxhToken, LPCSTR lpszServerAddr, LPCSTR lpszUser, LPCSTR lpszPass = NULL, BOOL bEnableHold = FALSE)

参数

 参数.一：pxhToken
  In/Out：Out
  类型：句柄
  可空：N
  意思：输出客户端句柄
 参数.二：lpszServerAddr
  In/Out：In
  类型：常量字符指针
  可空：N
  意思：输入服务器地址或者域名信息
 参数.二：lpszUser
  In/Out：In
  类型：常量字符指针
  可空：N
  意思：输入用户名
 参数.二：lpszPass
  In/Out：In
  类型：常量字符指针
  可空：Y
  意思：输入密码.如果密码为NULL,表示客户端只支持匿名登录
 参数.二：bEnableHold
  In/Out：In
  类型：常量字符指针
  可空：Y
  意思：是否启用穿透,需要服务器支持

返回值

类型：逻辑型
意思：是否成功

备注

注意,此模块不包含网络服务
