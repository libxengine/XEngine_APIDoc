函数

ProcFile_System_GetTime

功能

获取系统启动时间信息

语法

BOOL ProcFile_System_GetTime(__int64u* pInt_UPTime = NULL, __int64u* pInt_IdleTime = NULL)

参数

 参数.一：pInt_UPTime
  In/Out：Out
  类型：整数型指针
  可空：N
  意思：输出系统启动时间,单位秒
 参数.二：pInt_IdleTime
  In/Out：Out
  类型：整数型指针
  可空：N
  意思：输出系统启动来的空闲时间,单位秒

返回值

类型：逻辑型
意思：是否成功
