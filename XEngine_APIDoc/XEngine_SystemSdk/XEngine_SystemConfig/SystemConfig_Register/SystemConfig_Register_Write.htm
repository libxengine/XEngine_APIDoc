函数

SystemConfig_Register_Write

功能

写入注册表内容

语法

bool SystemConfig_Register_Write(XBYTE byType, LPCXSTR lpszKeyPath, LPCXSTR lpszKeyQuery, LPCXBTR lpszValue, int nSize, int nType)

参数

 参数.一：byType
  In/Out：In
  类型：注册表类型
  可空：N
  意思：0:HKEY_CLASSES_ROOT,1:HKEY_CURRENT_USER,2:HKEY_LOCAL_MACHINE,3:HKEY_USERS
 参数.二：lpszKeyPath
  In/Out：In
  类型：常量字符指针
  可空：N
  意思：注册表路径
 参数.三：lpszKeyQuery
  In/Out：In
  类型：常量字符指针
  可空：N
  意思：注册表项目
 参数.四：lpszValue
  In/Out：In
  类型：常量字符指针
  可空：N
  意思：输入写入的内容
 参数.五：nSize
  In/Out：Out
  类型：整数型
  可空：N
  意思：输入缓冲区大小
 参数.六：nType
  In/Out：In
  类型：整数型
  可空：N
  意思：输入类型.参考MSDN注册表值类型

返回值

类型：整数型
意思：是否成功
