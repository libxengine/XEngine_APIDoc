函数

SystemApi_File_SaveBuffToFile

功能

把缓冲区的内容保存为文件

语法

BOOL SystemApi_File_SaveBuffToFile(LPCSTR lpszFileName, LPCSTR lpszMsgBuffer, int nMsgLen)

参数

参数.一：lpszFileName
  In/Out：In
  类型：常量字符指针
  可空：N
  意思：保存文件路径
参数.二：lpszMsgBuffer
  In/Out：In
  类型：常量字符指针
  可空：N
  意思：缓冲区
参数.三：nMsgLen
  In/Out：In
  类型：整数型
  可空：N
  意思：缓冲区长度

返回值

类型：逻辑型
意思：是否成功
