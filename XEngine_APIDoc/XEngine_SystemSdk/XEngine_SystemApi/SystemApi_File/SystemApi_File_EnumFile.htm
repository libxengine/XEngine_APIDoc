函数

SystemApi_File_EnumFile

功能

枚举文件

语法

bool SystemApi_File_EnumFile(LPCXSTR lpszPath, XCHAR * **pppszListDir = NULL, int* pInt_ListCount = NULL, bool bRecursion = true, int nFindType = 3)

参数

 参数.一：lpszPath
  In/Out：In
  类型：常量字符指针
  可空：N
  意思：要枚举的文件目录
 参数.二：pppszListDir
  In/Out：Out
  类型：三级指针
  可空：Y
  意思：导出获取到的文件目录列表
 参数.三：pInt_ListCount
  In/Out：Out
  类型：整数型指针
  可空：Y
  意思：导出文件列表个数
 参数.四：bRecursion
  In/Out：In/Out
  类型：逻辑型
  可空：Y
  意思：是否枚举子目录，默认为真
 参数.五：nFindType
  In/Out：In
  类型：整数型
  可空：Y
  意思：查找的类型,1文件,2文件夹,3包含两者

返回值

类型：逻辑型
意思：是否成功

备注

参数二必须调用基础库的内存释放函数进行内存释放
