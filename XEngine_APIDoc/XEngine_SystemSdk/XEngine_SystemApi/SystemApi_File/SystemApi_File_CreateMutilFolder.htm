函数

SystemApi_File_CreateMutilFolder

功能

创建多级目录

语法

BOOL SystemApi_File_CreateMutilFolder(LPCSTR lpszFolder,DWORD dwFloder = 0)

参数

 参数.一：lpszFolder
   In/Out：In
   类型：常量字符指针
   可空：N
   意思：要创建的目录
 参数.二：dwFloder
   In/Out：In
   类型：双字
   可空：Y
   意思：要创建的目录的权限，如果指定0.那么表示任何权限的人都可以访问 读取 写入 执行

返回值

类型：逻辑型
意思：是否成功

备注

参数二必须调用基础库的内存释放函数进行内存释放
