函数

SystemApi_System_GetProcessList

功能

获取系统进程列表

语法

bool SystemApi_System_GetProcessList(SYSTEMAPI_PROCESS_INFO*** pppSt_ListProcess, int* pInt_ListCount)

参数

 参数.一：pppSt_ListProcess
  In/Out：Out
  类型：三级指针
  可空：N
  意思：输出获取到的进程列表,注意释放内存
 参数.二：pInt_ListCount
  In/Out：Out
  类型：整数型指针
  可空：N
  意思：输出获取到的进程数量

返回值

类型：逻辑型
意思：是否成功
