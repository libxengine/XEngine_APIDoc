函数

SystemApi_System_GetSystemVer

功能

获取系统版本和名称

语法

BOOL SystemApi_System_GetSystemVer(TCHAR * ptszOSName, TCHAR * ptszOSVersion, TCHAR * ptszOSBuild, DWORD * pInt_Processor, TCHAR * ptszKernelVer = NULL)

参数

 参数.一：pszOS
  In/Out：Out
  类型：字符指针
  可空：N
  意思：获取系统发行名称
 参数.二：pInt_OSVersion
  In/Out：Out
  类型：字符指针
  可空：N
  意思：获取发行版本号
 参数.三：ptszOSVersion
  In/Out：Out
  类型：字符指针
  可空：N
  意思：获取系统构建类型
 参数.四：ptszOSBuild
  In/Out：Out
  类型：字符指针
  可空：N
  意思：获取系统平台架构
 参数.五：ptszKernelVer
  In/Out：Out
  类型：字符指针
  可空：Y
  意思：获取内核版本号

返回值

类型：逻辑型
意思：是否成功
