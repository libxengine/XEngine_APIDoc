函数

SystemApi_System_GetSysName

功能

获取系统名称

语法

bool SystemApi_System_GetSysName(XCHAR* ptszUserName = NULL, XCHAR* ptszComputerName = NULL)

参数

 参数.一：ptszUserName
  In/Out：Out
  类型：字符指针
  可空：Y
  意思：输出系统用户名
 参数.二：ptszComputerName
  In/Out：Out
  类型：字符指针
  可空：Y
  意思：输出系统机器名

返回值

类型：逻辑型
意思：是否成功
