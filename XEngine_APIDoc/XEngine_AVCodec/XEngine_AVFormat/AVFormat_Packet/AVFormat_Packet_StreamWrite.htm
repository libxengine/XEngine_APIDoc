函数

AVFormat_Packet_StreamWrite

功能

写入媒体数据

语法

bool AVFormat_Packet_StreamWrite(XHANDLE xhNet, int nAVIndex, XHANDLE pSt_Packet, double* pdlAVTime = NULL, bool bWait = false, bool bChanged = false)

参数

 参数.一：xhNet
  In/Out：In
  类型：句柄
  可空：N
  意思：要操作的句柄
 参数.二：nAVIndex
  In/Out：In
  类型：整数型
  可空：N
  意思：数据的媒体索引
 参数.三：pSt_Packet
  In/Out：Out
  类型：句柄
  可空：N
  意思：输入要写的媒体数据
 参数.四：pdlAVTime
  In/Out：In
  类型：浮点型指针
  可空：Y
  意思：输出打包的媒体时间
 参数.五：bWait
  In/Out：In
  类型：逻辑型
  可空：Y
  意思：输入是否使用系统等待
 参数.六：bChanged
  In/Out：In
  类型：逻辑型
  可空：Y
  意思：这次输入是否是新的流(流切换)

返回值

类型：逻辑型
意思：是否成功

备注

切换流会导致pts时间戳变换,bChanged必须设置为真,否则会导致输出的流时间戳不连续
pSt_Packet支持设置NULL表示刷新内部写数据缓冲区,一般切换文件使用
