函数

AudioCodec_Stream_EnInit

功能

初始化流编解码器

语法

bool AudioCodec_Stream_EnInit(XNETHANDLE * pxhNet, AVCODEC_AUDIO_INFO * pSt_AudioInfo, __int64x nRateMin = 0, __int64x nRateMax = 0, int nRateLevel = -1)

参数

 参数.一：pxhNet
  In/Out：Out
  类型：句柄
  可空：N
  意思：导出初始化成功的音频编码句柄
 参数.二：pSt_AudioInfo
  In/Out：In
  类型：枚举型
  可空：Y
  意思：要编码成的音频格式
 参数.三：nRateMin
  In/Out：In
  类型：整数型
  可空：Y
  意思：编码器可变码率最小值,设置后变为可变码率
 参数.四：nRateMax
  In/Out：In
  类型：整数型
  可空：Y
  意思：编码器可变码率最大值,设置后变为可变码率
 参数.五：nRateLevel
  In/Out：In
  类型：整数型
  可空：Y
  意思：编码器码率等级,AAC支持,0为固定,1-5质量等级

返回值

类型：逻辑型
意思：是否成功

备注

部分编码格式需要指定一帧大小,比如G711...
