函数

AVHelp_Memory_GetVideoBuffer

功能

获取视频数据缓冲

语法

bool AVHelp_Memory_GetVideoBuffer(XHANDLE pSt_AVBuffer, XENGINE_MSGBUFFER* pSt_MSGBuffer, bool bAVPacket = true)

参数

 参数.一：pSt_AVBuffer
  In/Out：In
  类型：数据结构指针
  可空：N
  意思：输入要操作的原始数据结构
 参数.二：pSt_MSGBuffer
  In/Out：Out
  类型：数据结构指针
  可空：N
  意思：输出获取到的数据
 参数.二：bAVPacket
  In/Out：In
  类型：逻辑型
  可空：N
  意思：true是表示编码,false表示解码的数据结构

返回值

类型：逻辑型
意思：是否成功

备注

pSt_MSGBuffer缓冲区数据释放需要调用BaseLib_Memory_MSGFree
