函数

AVHelp_Memory_FreeAVList

功能

释放AVList列表帮助函

语法

bool AVHelp_Memory_FreeAVList(XHANDLE*** pppSt_AVBuffer, int nListCount, bool bAVPacket = true)

参数

 参数.一：pppSt_AVBuffer
  In/Out：In
  类型：三级指针
  可空：N
  意思：输入要释放的AVList列表数据
 参数.二：nListCount
  In/Out：In
  类型：整数型
  可空：N
  意思：输入列表个数
 参数.三：bAVPacket
  In/Out：In
  类型：逻辑型
  可空：N
  意思：true表示编码AVPacket,false表示解码AVFrame的数据结构

返回值

类型：逻辑型
意思：是否成功

备注

此函数可以帮助你释放AVList列表数据
