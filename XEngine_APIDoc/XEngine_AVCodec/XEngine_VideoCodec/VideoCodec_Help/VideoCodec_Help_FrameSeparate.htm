函数

VideoCodec_Help_FrameSeparate

功能

分离原始帧数据

语法

bool VideoCodec_Help_FrameSeparate(int nPIXFormat, int nWidth, int nHeight, uint8_t* ptszMSGBuffer, AVCODEC_VIDEO_MSGBUFFER st_MSGBuffers[4], int* pInt_ListCount)

参数

 参数.一：nPIXFormat
  In/Out：In
  类型：整数型
  可空：N
  意思：视频采样格式
 参数.二：nWidth
  In/Out：In
  类型：整数型
  可空：N
  意思：视频的宽
 参数.三：nHeight
  In/Out：In
  类型：整数型
  可空：N
  意思：视频的高
 参数.四：ptszMSGBuffer
  In/Out：In
  类型：字符指针
  可空：N
  意思：输入要分离的数据
 参数.五：st_MSGBuffers
  In/Out：Out
  类型：数据结构数组
  可空：N
  意思：输出分离的数据
 参数.六：pInt_ListCount
  In/Out：Out
  类型：整数型指针
  可空：N
  意思：分离的数据个数

返回值

类型：逻辑型
意思：是否成功
