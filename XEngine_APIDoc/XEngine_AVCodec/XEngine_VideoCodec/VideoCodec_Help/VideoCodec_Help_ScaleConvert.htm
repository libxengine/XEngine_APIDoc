函数

VideoCodec_Help_ScaleConvert

功能

转换一帧图像数据

语法

bool VideoCodec_Help_ScaleConvert(XHANDLE xhToken, XHANDLE pSt_SrcFrame, XHANDLE*** pppSt_DstFrame, int nSliceY = 0, int nSliceH = 0)

参数

 参数.一：xhToken
  In/Out：In
  类型：句柄
  可空：N
  意思：输入要操作的转换器
 参数.二：pSt_SrcFrame
  In/Out：In
  类型：句柄
  可空：N
  意思：原始图像格式帧
 参数.三：pppSt_DstFrame
  In/Out：Out
  类型：三级指针
  可空：N
  意思：输出转换后的目标格式
 参数.四：nSliceY
  In/Out：In
  类型：整数型
  可空：Y
  意思：要开始转换的Y坐标点
 参数.五：nSliceH
  In/Out：In
  类型：整数型
  可空：Y
  意思：转换的行

返回值

类型：逻辑型
意思：是否成功
