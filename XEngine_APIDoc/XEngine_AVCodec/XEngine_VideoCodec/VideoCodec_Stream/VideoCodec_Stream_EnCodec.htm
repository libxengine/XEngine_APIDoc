函数

VideoCodec_Stream_EnCodec

功能

编码图像

语法

bool VideoCodec_Stream_EnCodec(XNETHANDLE xhNet, uint8_t* ptszAVBuffer, int nAVLen, AVCODEC_VIDEO_MSGBUFFER*** pppSt_MSGBuffer, int* pInt_ListCount, __int64u nPTS = 0, __int64u nDTS = 0, bool bKeyFrame = false)

参数

 参数.一：xhNet
  In/Out：In
  类型：句柄
  可空：N
  意思：要操作的编码器句柄
 参数.二：ptszAVBuffer
  In/Out：In
  类型：无符号整数型指针
  可空：N
  意思：原始帧的数据缓冲区
 参数.三：nAVLen
  In/Out：In
  类型：整数型
  可空：N
  意思：数据缓冲区大小
 参数.四：pppSt_MSGBuffer
  In/Out：Out
  类型：三级指针
  可空：N
  意思：导出编码好的数据结构
 参数.五：pInt_ListCount
  In/Out：In/Out
  类型：整数型指针
  可空：N
  意思：输出编码的个数
 参数.六：nPTS
  In/Out：In
  类型：整数型
  可空：Y
  意思：自定义PTS
 参数.七：nDTS
  In/Out：In
  类型：整数型
  可空：Y
  意思：自定义DTS,有B帧的情况需要
 参数.八：bKeyFrame
  In/Out：In
  类型：逻辑型
  可空：Y
  意思：是否编码成关键帧

返回值

类型：逻辑型
意思：是否成功

备注

nYLen = 0,表示发送结束帧
