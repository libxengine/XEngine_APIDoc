函数

APIClient_Http_Excute

功能

执行一个请求

语法

bool APIClient_Http_Excute(XNETHANDLE xhToken, LPCXSTR lpszCustomBody = XNULL, int nMSGLen = 0, LPCXSTR lpszCustomHdr = NULL)

参数

 参数.一：xhToken
  In/Out：In
  类型：句柄
  可空：N
  意思：输入要操作的HTTP句柄
 参数.二：lpszCustomBody
  In/Out：In
  类型：常量字符指针
  可空：Y
  意思：输入请求的附加数据
 参数.三：nMSGLen
  In/Out：In
  类型：整数型
  可空：Y
  意思：输入负载内容数据大小
 参数.四：lpszCustomHdr
  In/Out：In
  类型：常量字符指针
  可空：Y
  意思：输入请求的自定义头\r\n分割

返回值

类型：逻辑型
意思：是否成功

备注
