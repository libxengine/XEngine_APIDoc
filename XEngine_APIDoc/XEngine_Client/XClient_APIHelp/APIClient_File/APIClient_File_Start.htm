函数

APIClient_File_Start

功能

开始下载或者上传

语法

BOOL APIClient_File_Start(XHANDLE xhDown, BOOL bIsPasv = FALSE, LPCTSTR lpszMethod = NULL)

参数

 参数.一：xhDown
  In/Out：In
  类型：句柄
  可空：N
  意思：要处理的任务句柄
 参数.二：bIsPasv
  In/Out：In
  类型：逻辑型
  可空：N
  意思：FTP传输模式
 参数.三：lpszMethod
  In/Out：In
  类型：常量字符指针
  可空：Y
  意思：输入自定义操作方法

返回值

类型：句柄型
意思：是否成功
