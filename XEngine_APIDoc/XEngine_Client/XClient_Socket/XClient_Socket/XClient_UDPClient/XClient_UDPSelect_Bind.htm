函数

XClient_UDPSelect_Bind

功能

绑定端口

语法

BOOL XClient_UDPSelect_Bind(SOCKET hSocket, int nPort, LPCSTR lpszAddr = NULL, int nIPVer = 2)

参数

 参数.一：hSocket
  In/Out：In
  类型：套接字句柄
  可空：N
  意思：要操作的UDP客户端
 参数.二：nPort
  In/Out：In
  类型：整数型
  可空：N
  意思：要绑定的端口
 参数.三：lpszAddr
  In/Out：In
  类型：常量字符指针
  可空：Y
  意思：要绑定的网卡IP地址
 参数.四：nIPVer
  In/Out：In
  类型：整数型
  可空：Y
  意思：要使用的IP版本,V4或者V6

返回值

类型：逻辑型
意思：是否成功
