函数

XClient_TCPSelect_GetFlowEx

功能

获取流量信息

语法

BOOL XClient_TCPSelect_GetFlowEx(XHANDLE xhNet, __int64u* pInt_SendPkt = NULL, __int64u* pInt_SendByte = NULL, __int64u* pInt_RecvPkt = NULL, __int64u* pInt_RecvByte = NULL)

参数

 参数.一：xhNet
  In/Out：In
  类型：句柄
  可空：N
  意思：输入要获取的客户端
 参数.二：pInt_SendPkt
  In/Out：Out
  类型：整数型指针
  可空：Y
  意思：输出发送的包个数
 参数.三：pInt_SendByte
  In/Out：Out
  类型：整数型指针
  可空：Y
  意思：输出发送的大小
 参数.四：pInt_RecvPkt
  In/Out：Out
  类型：整数型指针
  可空：Y
  意思：输出接受的包个数
 参数.五：pInt_RecvByte
  In/Out：Out
  类型：整数型指针
  可空：Y
  意思：输出接受的大小

返回值

类型：逻辑型
意思：是否成功
