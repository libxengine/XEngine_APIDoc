函数

XClient_StreamPush_LiveVideo

功能

推送一个视频数据

语法

bool XClient_StreamPush_LiveVideo(XHANDLE xhNet, uint8_t* ptszAVBuffer, int nAVLen, bool bKeyFrame = false, __int64x nPTSValue = 0, __int64x nDTSValue = 0, __int64x nDuration = 0)

参数

 参数.一：xhNet
  In/Out：In
  类型：句柄
  可空：N
  意思：输入初始化的推流句柄
 参数.二：ptszAVBuffer
  In/Out：In
  类型：无符号整数型指针
  可空：N
  意思：要处理的缓冲区
 参数.三：nAVLen
  In/Out：In
  类型：整数型
  可空：N
  意思：缓冲区大小
 参数.四：bKeyFrame
  In/Out：In
  类型：逻辑型
  可空：Y
  意思：是否设置关键帧编码
 参数.五：nPTSValue
  In/Out：In
  类型：整数型
  可空：Y
  意思：自定义PTS时间戳
 参数.六：nDTSValue
  In/Out：In
  类型：整数型
  可空：Y
  意思：自定义DTS时间戳
 参数.七：nDuration
  In/Out：In
  类型：整数型
  可空：Y
  意思：真持续时间

返回值

类型：逻辑型
意思：是否成功
