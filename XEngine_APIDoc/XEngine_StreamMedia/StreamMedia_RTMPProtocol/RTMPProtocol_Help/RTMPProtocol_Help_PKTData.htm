函数

RTMPProtocol_Help_PKTData

功能

打包数据协议

语法

bool RTMPProtocol_Help_PKTData(XCHAR* ptszMsgBuffer, int* pInt_MsgLen, XBYTE byChunkID, XENGINE_RTMPCMDPROPERTY*** pppSt_PropertyList, int nListCount)

参数

 参数.一：ptszMsgBuffer
  In/Out：Out
  类型：字符指针
  可空：N
  意思：输出打包好的协议缓冲区
 参数.二：pInt_MsgLen
  In/Out：Out
  类型：整数型指针
  可空：N
  意思：输出缓冲区大小
 参数.三：byChunkID
  In/Out：In
  类型：字符
  可空：N
  意思：输入要打包的命令ID
 参数.四：pppSt_PropertyList
  In/Out：In
  类型：三级指针
  可空：N
  意思：要打包的数据
 参数.五：nListCount
  In/Out：In
  类型：整数型
  可空：N
  意思：打包数据个数

返回值

类型：逻辑型
意思：是否成功
