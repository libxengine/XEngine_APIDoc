函数

RTSPProtocol_REPParse_Request

功能

解析缓冲区

语法

bool RTSPProtocol_REPParse_Request(RTSPPROTOCOL_RESPONSE * pSt_RTSPReponse, LPCXSTR lpszMsgBuffer, int nMsgLen)

参数

 参数.一：pSt_RTSPReponse
  In/Out：Out
  类型：数据结构指针
  可空：N
  意思：输出解析出来的RTSP回复
 参数.二：lpszMsgBuffer
  In/Out：In
  类型：常量字符指针
  可空：N
  意思：输入要解析的缓冲区
 参数.三：nMsgLen
  In/Out：In
  类型：整数型
  可空：N
  意思：输入缓冲区大小

返回值

类型：逻辑型
意思：是否成功
