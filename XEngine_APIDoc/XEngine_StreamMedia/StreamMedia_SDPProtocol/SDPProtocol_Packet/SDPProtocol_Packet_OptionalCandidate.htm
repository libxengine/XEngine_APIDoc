函数

SDPProtocol_Packet_OptionalCandidate

功能

WEBRTC的网络连接选项.WEBRTC必带

语法

bool SDPProtocol_Packet_OptionalCandidate(XNETHANDLE xhToken, LPCXSTR lpszIPAddr, int nIPPort, XBYTE byICEType = 0, XBYTE byTransProtocol = 0, XBYTE byComponentsID = 1)

参数

 参数.一：xhToken
  In/Out：In
  类型：句柄
  可空：N
  意思：输入要操作的SDP会话
 参数.二：lpszIPAddr
  In/Out：In
  类型：常量字符指针
  可空：N
  意思：输入服务IP地址
 参数.三：nIPPort
  In/Out：In
  类型：整数型
  可空：N
  意思：输入通信端口
 参数.四：byICEType
  In/Out：In
  类型：字符型
  可空：Y
  意思：输入ICE类型,0为HOST(常用,物理直链),1为srflx(STUN获取NAT转发),2为prflx(通信检测),3为relay(中继候选)
 参数.五：byTransProtocol
  In/Out：In
  类型：字符型
  可空：Y
  意思：传输协议,0为UDP传输,1为TCP
 参数.六：byComponentsID
  In/Out：In
  类型：字符型
  可空：Y
  意思：组件ID,1为RTP,2为RTCP

返回值

类型：逻辑型
意思：是否成功

备注

可选函数,可添加多次
