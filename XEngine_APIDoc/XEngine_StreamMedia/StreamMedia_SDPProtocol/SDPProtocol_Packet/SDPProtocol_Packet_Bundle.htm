函数

SDPProtocol_Packet_Bundle

功能

打包流绑定属性

语法

bool SDPProtocol_Packet_Bundle(XNETHANDLE xhToken, int nIndex1 = 0, int nIndex2 = 1)

参数

 参数.一：xhToken
  In/Out：In
  类型：句柄
  可空：N
  意思：输入要操作的SDP会话
 参数.二：nIndex1
  In/Out：In
  类型：整数型
  可空：Y
  意思：输入流的MID索引,-1不启用
 参数.三：nIndex2
  In/Out：In
  类型：整数型
  可空：Y
  意思：输入流的MID索引,-1不启用

返回值

类型：逻辑型
意思：是否成功
