函数

APIClient_Http_GetResult

功能

获取结果

语法

bool APIClient_Http_GetResult(XNETHANDLE xhToken, bool* pbComplete, int* pInt_HTTPCode = NULL)

参数

 参数.一：xhToken
  In/Out：In
  类型：句柄
  可空：N
  意思：要操作的HTTP客户端
 参数.二：pbComplete
  In/Out：Out
  类型：逻辑型指针
  可空：N
  意思：输出是否执行完毕
 参数.三：pInt_HTTPCode
  In/Out：Out
  类型：整数型指针
  可空：Y
  意思：输出得到的HTTP状态码

返回值

类型：逻辑型
意思：是否成功

备注
