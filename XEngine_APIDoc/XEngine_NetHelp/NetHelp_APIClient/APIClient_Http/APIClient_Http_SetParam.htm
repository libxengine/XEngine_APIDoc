函数

APIClient_Http_SetParam

功能

设置HTTP参数

语法

BOOL APIClient_Http_SetParam(XNETHANDLE xhToken, APIHELP_HTTPPARAMENT * pSt_HTTPParam)

参数

 参数.一：xhToken
  In/Out：In
  类型：句柄
  可空：N
  意思：输入要操作的HTTP句柄
 参数.二：pSt_HTTPParam
  In/Out：In
  类型：数据结构指针
  可空：N
  意思：输入HTTP参数

返回值

类型：逻辑型
意思：是否成功

备注

按照服务器要求设置请求
