函数

XSocket_Api_GetAddress

功能

获取套接字的IP地址

语法

BOOL XSocket_Api_GetAddress(SOCKET hSocket, CHAR* ptszIPAddr, BOOL bLocal = TRUE, int nIPVer = AF_INET)

参数

 参数.一：hSocket
  In/Out：In
  类型：套接字句柄
  可空：N
  意思：输入要操作的套接字
 参数.二：ptszIPAddr
  In/Out：Out
  类型：字符指针
  可空：N
  意思：输出IP:PORT
 参数.三：bLocal
  In/Out：In
  类型：逻辑型
  可空：Y
  意思：是获取本地还是远程,默认本地
 参数.四：nIPVer
  In/Out：In
  类型：整数型
  可空：Y
  意思：输入IP版本

返回值

类型：逻辑型
意思：是否成功
