函数

Cryption_Cert_X509Verifly

功能

验证证书是否正确

语法

BOOL Cryption_Cert_X509Verifly(LPCSTR lpszCARoot, LPCSTR lpszCAUser, int nCertType = XENGINE_OPENSSL_OPENSSL_PEM_FILE)

参数

 参数.一：lpszCARoot
  In/Out：In
  类型：常量字符指针
  可空：N
  意思：根证书保存路径
 参数.二：lpszCAUser
  In/Out：In
  类型：常量字符指针
  可空：N
  意思：个人证书路径
 参数.三：nCertType
  In/Out：In
  类型：整数型
  可空：Y
  意思：证书类型,PEM还是DER

返回值

类型：逻辑型
意思：证书是否验证成功
