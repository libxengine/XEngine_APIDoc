函数

Cryption_Api_RSAPubFile

功能

通过私钥文件创建公钥文件函数

语法

bool Cryption_Api_RSAPubFile(LPCXSTR lpszPriFile, LPCXSTR lpszPubFile, LPCXSTR lpszPassStr = NULL)

参数

 参数.一：lpszPriFile
  In/Out：In
  类型：常量字符指针
  可空：N
  意思：读取私钥文件路径
 参数.二：lpszPubFile
  In/Out：In
  类型：常量字符指针
  可空：N
  意思：写入公钥文件路径
 参数.三：lpszPassStr
  In/Out：In
  类型：常量字符指针
  可空：Y
  意思：私钥的密码

返回值

类型：逻辑型
意思：是否生成成功
