函数

Cryption_Cert_GetCerInfomachine

功能

获取个人证书信息

语法

BOOL Cryption_Cert_GetCerInfomachine(LPCSTR lpszCerFile, LPOPENSSL_X509CCINFO pSt_X509CCInfo)

参数

 参数.一：lpszCerFile
  In/Out：In
  类型：常量字符指针
  可空：N
  意思：要查询的证书路径
 参数.二：pSt_X509CCInfo
  In/Out：In
  类型：结构体指针
  可空：N
  意思：输出个人证书所查询到的信息

返回值

类型：逻辑型
意思：是否查询成功

备注

 

示例

void GetCert()
{
#ifdef _WINDOWS
    LPCTSTR lpszKey = _T("H:\\XEngine_Apps\\Debug\\test.crt");
#else
    LPCTSTR lpszKey = _T("test.crt");
#endif
    OPENSSL_X509CCINFO st_X509Info;
    memset(&st_X509Info, '\0', sizeof(OPENSSL_X509CCINFO));

    if (!OPenSsl_Cert_GetCerInfomachine(lpszKey, &st_X509Info))
    {
        return;
    }
}
