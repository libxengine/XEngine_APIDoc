函数

Cryption_Server_AcceptEx

功能

接受一个SSL连接

语法

bool Cryption_Server_AcceptEx(XHANDLE xhToken, XSOCKET hSocket, LPCXSTR lpszClientAddr, XCHAR * ptszClientAddr = NULL)

参数

 参数.一：hSocket
  In/Out：In
  类型：网络套接字
  可空：N
  意思：输入你的ACCEPT返回的套接字句柄
 参数.二：lpszClientAddr
  In/Out：In
  类型：常量字符指针
  可空：N
  意思：输入你的客户端地址的信息IP:PORT
 参数.三：ptszClientAddr
  In/Out：Out
  类型：字符指针
  可空：Y
  意思：得到客户端地址信息,如果lpszClientAddr为NULL,那么此参数有效

返回值

类型：逻辑型
意思：是否成功

备注
