函数

NetCore_UDXSocket_InitEx 

功能

初始化UDX

语法

XHANDLE NetCore_UDXSocket_InitEx(NETCORE_UDXCONFIG *pSt_UDXConfig, int nPort, int nIPVer = 2)

参数

 参数.一：pSt_UDXConfig
  In/Out：In
  类型：数据结构指针
  可空：N
  意思：设置UDX的服务器选项
 参数.二：nPort
  In/Out：In
  类型：整数型
  可空：N
  意思：要绑定的端口号
 参数.三：nIPVer
  In/Out：In
  类型：整数型
  可空：Y
  意思：输入要使用的IP协议版本

返回值

类型：句柄型
意思：返回创建成功的句柄

备注

示例

int test_udpxcore()
{
    XHANDLE xhUDXCore = NetCore_UDPXCore_StartEx(&xhUDPCore, 5002);
    if (NULL != xhUDXCore)
    {
        printf(_T("NetCore_UDPXCore_StartEx Start Is Ok!\n"));
    }
    else
    {
        printf(_T("NetCore_UDPXCore_StartEx Start Is Failed!\n"));
    }
    NetCore_UDPXCore_RegisterCallBackEx(xhUDXCore, Callback_UDPRecv);
    return 0;
}
