函数

SocketOpt_HeartBeat_ActiveSocketEx

功能

BOOL SocketOpt_HeartBeat_ActiveSocketEx(XHANDLE xhNet, SOCKET hSocket, XENGINE_PROTOCOL_HEARTBEAT *pSt_ProtocolHeartBeat = NULL)

参数

 参数.一：hSocket
  In/Out：In
  类型：套接字句柄
  可空：N
  意思：要激活的客户端，唯一标识符
 参数.二：pSt_ProtocolHeartBeat
  In/Out：In
  类型：数据结构指针
  可空：Y
  意思：心跳协议

返回值

类型：逻辑型
意思：是否成功
