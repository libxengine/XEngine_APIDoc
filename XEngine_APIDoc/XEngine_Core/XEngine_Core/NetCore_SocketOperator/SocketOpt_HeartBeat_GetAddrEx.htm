函数

SocketOpt_HeartBeat_GetAddrEx

功能

获取一个地址信息

语法

BOOL SocketOpt_HeartBeat_GetAddrEx(XHANDLE xhNet, LPCSTR lpszClientAddr, __int64x* pInt_Timer = NULL)

参数

 参数.一：lpszClientAddr
  In/Out：In
  类型：常量字符指针
  可空：N
  意思：输入要判断的某一个地址
 参数.二：pInt_Timer
  In/Out：Out
  类型：整数型指针
  可空：Y
  意思：输出在线时长,单位秒

返回值

类型：逻辑型
意思：是否成功

备注

此函数可以判断一个地址是否存在,返回真表示存在
