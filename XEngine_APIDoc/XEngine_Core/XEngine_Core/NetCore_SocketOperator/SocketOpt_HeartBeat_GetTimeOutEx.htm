函数

SocketOpt_HeartBeat_GetTimeOutEx 

功能

从心跳管理中获取一个超时的客户端标识符

语法

BOOL SocketOpt_HeartBeat_GetTimeOutEx(XHANDLE xhNet, CHAR *ptszClientAddr = NULL, SOCKET *phSocket = NULL)

参数

 参数.一：ptszClientAddr
  In/Out：Out
  类型：字符指针
  可空：Y
  意思：获取到的唯一标识符
 参数.二：phSocket
  In/Out：Out
  类型：套接字句柄
  可空：Y
  意思：获取到的唯一标识符

返回值

类型：逻辑型
意思：是否成功

备注

如果设置了回调函数，这个函数将不起作用，此方式你需要手动调用函数删除超时客户端
