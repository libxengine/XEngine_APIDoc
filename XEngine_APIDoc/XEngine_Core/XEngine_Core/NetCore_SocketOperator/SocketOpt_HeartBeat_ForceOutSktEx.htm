函数

SocketOpt_HeartBeat_ForceOutSktEx

功能

强制让一个客户端心跳超时

语法

BOOL SocketOpt_HeartBeat_ForceOutSktEx(XHANDLE xhNet, SOCKET hSocket, int nStatus = 0, BOOL bIgnore = FALSE)

参数

 参数.一：hSocket
  In/Out：In
  类型：套接字句柄
  可空：N
  意思：要超时的客户端句柄
 参数.二：nStatus
  In/Out：In
  类型：逻辑型
  可空：Y
  意思：强制退出用户信息状态
 参数.三：bIgnore
  In/Out：In
  类型：逻辑型
  可空：Y
  意思：是否忽略这个地址是否在心跳管理器中，默认不忽略

返回值

类型：逻辑型
意思：是否成功

备注

在某些极端情况下，开发人员可能需要这样的功能
