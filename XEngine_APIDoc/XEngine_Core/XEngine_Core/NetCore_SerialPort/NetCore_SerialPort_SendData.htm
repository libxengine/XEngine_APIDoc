函数

NetCore_SerialPort_SendData 

说明

异步发送串口数据 

语法

BOOL NetCore_SerialPort_SendData(LPCSTR lpszComPort,BYTE *byDates,int nWriteLen)

参数

 参数.一：lpszComPort
  In/Out：In
  类型：常量字符指针
  可空：N
  意思：要发送的数据字符串指针
 参数.二：byDates
  In/Out：In
  类型：无符号字节型指针
  可空：N
  意思：要发送的数据
 参数.三：nWriteLen
  In/Out：In
  类型：整数型
  可空：N
  意思：要发送的长度

返回值

类型：逻辑型
意思：是否发送成功

备注

如果错误你需要获取错误码来知道为什么错误
