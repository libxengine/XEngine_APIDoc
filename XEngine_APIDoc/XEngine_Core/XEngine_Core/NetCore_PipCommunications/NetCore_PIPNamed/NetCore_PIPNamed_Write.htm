函数

NetCore_PIPNamed_Write

说明

写入数据到指定的命名管道

语法

BOOL NetCore_PIPNamed_Write(LPCSTR lpszPipName, LPCSTR lpszMsgBuffer, int nLen);

参数

 参数.一：lpszHostName
  In/Out：In
  类型：常量字符指针
  可空：N
  意思：要写入到哪个命名管道中
 参数.二：lpszMsgBuffer
  In/Out：In
  类型：常量字符指针
  可空：N
  意思：写入到的数据缓冲区
 参数.三：nLen
  In/Out：In
  类型：整数型
  可空：N
  意思：要写入的大小

返回值

类型：逻辑型
意思：是否成功写入数据

备注
