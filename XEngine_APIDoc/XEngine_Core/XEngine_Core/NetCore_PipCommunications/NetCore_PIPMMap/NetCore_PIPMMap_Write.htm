函数

NetCore_PIPMMap_Write

说明

写入数据到内存映射中

语法

BOOL NetCore_PIPMMap_Write(LPCSTR lpszMapName, LPVOID lpbBuffer, int nLen, int nPos = 0);

参数

 参数.一：lpszMapName
  In/Out：In
  类型：常量字符指针
  可空：N
  意思：要写入到哪个内存映射中
 参数.二：lpVoid_Buffer
  In/Out：In
  类型：无类型指针
  可空：N
  意思：要写入的数据
 参数.三：nLen
  In/Out：In
  类型：整数型
  可空：N
  意思：写入数据的长度
 参数.四：nPos
  In/Out：In
  类型：整数型
  可空：Y
  意思：写入的位置

返回值

类型：逻辑型
意思：是否成功写入数据到内存映射中

备注
