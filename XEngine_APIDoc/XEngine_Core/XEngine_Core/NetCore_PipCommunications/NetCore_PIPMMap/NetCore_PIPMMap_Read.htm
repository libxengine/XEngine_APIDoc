函数

NetCore_PIPMMap_Read

说明

从内存映射中读取数据

语法

BOOL NetCore_PIPMMap_Read(LPCSTR lpszMapName, LPVOID lpbBuffer, int nLen, int nPos = 0)

参数

 参数.一：lpszMapName
  In/Out：In
  类型：常量字符指针
  可空：N
  意思：要读取的内存映射名称
 参数.二：lpbBuffer
  In/Out：Out
  类型：无类型指针
  可空：N
  意思：读取到的数据
 参数.三：nLen
  In/Out：In
  类型：整数型
  可空：N
  意思：要读取的大小
 参数.四：nPos
  In/Out：In
  类型：整数型
  可空：Y
  意思：要读取的位置

返回值

类型：逻辑型
意思：是否读取成功

备注
