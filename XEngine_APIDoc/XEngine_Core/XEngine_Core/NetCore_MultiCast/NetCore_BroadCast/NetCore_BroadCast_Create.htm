函数

NetCore_BroadCast_Create

说明

初始化广播端

语法

BOOL NetCore_BroadCast_Create(SOCKET *phSocket,int nPort,LPCSTR lpszAddr = NULL)

参数

 参数.一：phSocket
  In/Out：Out
  类型：套接字指针
  可空：N
  意思：导出的套接字
 参数.二：nPort
  In/Out：In
  类型：整数型
  可空：N
  意思：要发送端口
 参数.三：lpszAddr
  In/Out：In
  类型：常量字符指针
  可空：Y
  意思：如果有多张网卡，那么需要指定哪张网卡IP地址发送广播包

返回值

类型：逻辑型
意思：是否创建成功

备注
