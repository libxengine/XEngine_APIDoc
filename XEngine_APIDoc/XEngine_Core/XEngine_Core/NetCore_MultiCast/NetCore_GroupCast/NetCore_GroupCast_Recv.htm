函数

NetCore_GroupCast_Recv

说明

发送消息

语法

BOOL BOOL NetCore_GroupCast_Recv(SOCKET hSocket, TCHAR* ptszMsgBuffer, int* pInt_Len, TCHAR* ptszClientAddr = NULL, int nTimeout = 100)

参数

 参数.一：hSocket
  In/Out：In
  类型：网络套接字
  可空：N
  意思：操作哪个组播套接字
 参数.二：ptszMsgBuffer
  In/Out：In/Out
  类型：字符指针
  可空：N
  意思：要接受的数据缓冲区
 参数.三：pInt_Len
  In/Out：In/Out
  类型：整数型指针
  可空：N
  意思：输入提供的缓冲区大小，输出接受到的数据缓冲区大小
 参数.四：ptszClientAddr
  In/Out：Out
  类型：字符指针
  可空：Y
  意思：导出发送者的IP地址端口
 参数.五：nTimeout
  In/Out：In
  类型：整数型
  可空：Y
  意思：超时时间,-1不超时,0立即返回,>0 等待毫秒

返回值

类型：逻辑型
意思：是否成功

备注
