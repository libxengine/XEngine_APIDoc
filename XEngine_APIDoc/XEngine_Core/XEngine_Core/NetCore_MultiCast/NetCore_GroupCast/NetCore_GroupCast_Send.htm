函数

NetCore_GroupCast_Send

说明

发送消息

语法

BOOL NetCore_GroupCast_Send(SOCKET hSocket, LPCTSTR lpszMsgBuffer, int nLen)

参数

 参数.一：hSocket
  In/Out：In
  类型：网络套接字
  可空：N
  意思：操作哪个组播套接字
 参数.二：lpszMsgBuffer
  In/Out：In
  类型：常量字符指针
  可空：N
  意思：发送的消息
 参数.三：nLen
  In/Out：In
  类型：整数型
  可空：N
  意思：发送消息的长度

返回值

类型：逻辑型
意思：是否成功

备注
