函数

NetCore_TCPXCore_PasueRecvEx

说明

暂停或者开始一个指定客户端接受数据

语法

bool NetCore_TCPXCore_PasueRecvEx(XHANDLE xhNet, LPCXSTR lpszClientAddr, bool bRecv = true)

参数

 参数.一：lpszClientAddr
  In/Out：In
  类型：常量字符指针
  可空：N
  意思：输入要操作的客户端
 参数.二：bRecv
  In/Out：In
  类型：逻辑型
  可空：Y
  意思：真为开始接受,假为暂停接受

返回值

类型：句柄
意思：是否启动成功

备注
