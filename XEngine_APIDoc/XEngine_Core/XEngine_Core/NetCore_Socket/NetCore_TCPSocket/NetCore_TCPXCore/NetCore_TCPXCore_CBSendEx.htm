函数

NetCore_TCPXCore_CBSendEx

说明

设置发送可写回调

语法

BOOL NetCore_TCPXCore_CBSendEx(XHANDLE xhNet, LPCSTR lpszClientAddr, CALLBACK_NETCORE_SOCKET_NETEVENT_SEND fpCall_EventSend = NULL, LPVOID lParam = NULL)

参数

 参数.一：lpszClientAddr
  In/Out：In
  类型：常量字符指针
  可空：N
  意思：输入要操作的客户端
 参数.二：fpCall_EventSend
  In/Out：In/Out
  类型：回调函数
  可空：Y
  意思：当数据可写的时候,触发一次回调,为NULL表示取消回调发送
 参数.三：lParam
  In/Out：In/Out
  类型：无类型指针
  可空：Y
  意思：回调函数自定义参数

返回值

类型：逻辑型
意思：是否成功

备注

触发回调后调用NetCore_TCPXCore_SendMsg 函数发送数据
