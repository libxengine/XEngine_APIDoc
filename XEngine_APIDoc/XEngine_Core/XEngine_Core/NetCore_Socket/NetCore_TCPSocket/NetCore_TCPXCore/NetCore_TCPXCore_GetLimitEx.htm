函数

NetCore_TCPXCore_GetLimitEx

说明

获取服务限制

语法

BOOL NetCore_TCPXCore_GetLimitEx(XHANDLE xhNet, LPCSTR lpszClientAddr, int* pInt_RVMax, CHAR* ptszFindStr = NULL)

参数

 参数.一：lpszClientAddr
  In/Out：In
  类型：常量字符指针
  可空：N
  意思：输入要获取的客户端
 参数.二：pInt_RVMax
  In/Out：Out
  类型：整数型指针
  可空：Y
  意思：输出客户端设置的限制信息
 参数.三：ptszFindStr
  In/Out：In
  类型：字符指针
  可空：Y
  意思：输出设置的匹配字符串

返回值

类型：逻辑型
意思：是否成功

备注
