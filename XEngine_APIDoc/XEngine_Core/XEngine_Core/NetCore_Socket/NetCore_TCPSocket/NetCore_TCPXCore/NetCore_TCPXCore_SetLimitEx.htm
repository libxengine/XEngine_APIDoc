函数

NetCore_TCPXCore_SetLimitEx

说明

设置服务限制

语法

BOOL NetCore_TCPXCore_SetLimitEx(XHANDLE xhNet, LPCSTR lpszClientAddr, int nRVMax = 0, LPCSTR lpszFindStr = NULL)

参数

 参数.一：lpszClientAddr
  In/Out：In
  类型：常量字符指针
  可空：N
  意思：输入要设置的客户端
 参数.二：nRVMax
  In/Out：In
  类型：整数型
  可空：Y
  意思：输入客户端输入缓冲区大小,0没有限制(只有到达设置大小才会触发回调函数)
 参数.三：lpszFindStr
  In/Out：In
  类型：常量字符指针
  可空：Y
  意思：输入匹配触发回调的字符串,此参数必须设置参数二

返回值

类型：逻辑型
意思：是否成功

备注
