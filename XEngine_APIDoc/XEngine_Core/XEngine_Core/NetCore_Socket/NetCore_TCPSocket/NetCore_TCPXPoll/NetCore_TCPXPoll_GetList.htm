函数

NetCore_TCPXPoll_GetList

说明

获取客户端列表

语法

BOOL NetCore_TCPXPoll_GetList(TCHAR*** ppptszListAddr, int* pInt_ListCount)

参数

 参数.一：ppptszListAddr
  In/Out：Out
  类型：三级指针
  可空：N
  意思：输出客户端列表
 参数.二：pInt_ListCount
  In/Out：Out
  类型：整数型指针
  可空：N
  意思：输出列表个数

返回值

类型：逻辑型
意思：是否成功

备注

如果参数一为NULL,那么就只统计客户端数量
