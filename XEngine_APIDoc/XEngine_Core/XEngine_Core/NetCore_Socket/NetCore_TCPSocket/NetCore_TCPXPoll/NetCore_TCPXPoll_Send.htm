函数

NetCore_TCPXPoll_Send

说明

为指定客户发送数据

语法

BOOL NetCore_TCPXPoll_Send(LPCSTR lpszClientAddr,LPCSTR lpszMsgBuffer,int nMsgLen)

参数

 参数.一：lpszClientAddr
  In/Out：In
  类型：常量字符指针
  可空：N
  意思：发送的客户地址
 参数.二：lpszMsgBuffer
  In/Out：In
  类型：常量字符指针
  可空：N
  意思：要发送的数据缓冲区地址
 参数.三：nMsgLen
  In/Out：In
  类型：整数型
  可空：N
  意思：输入发送大小

返回值

类型：逻辑型
意思：是否启动成功

备注
