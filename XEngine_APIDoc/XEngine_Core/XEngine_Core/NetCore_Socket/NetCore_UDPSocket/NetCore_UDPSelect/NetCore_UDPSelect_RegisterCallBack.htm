函数

NetCore_UDPSelect_RegisterCallBack

说明

设置接受数据回调函数

语法

BOOL NetCore_UDPSelect_RegisterCallBack(XHANDLE xhNet, CALLBACK_NETCORE_SOCKET_NETEVENT_RECV fpCall_UDPEvent = NULL, LPVOID lParam = NULL);

参数

 参数.一：fpCall_UDPEvent
  In/Out：In/Out
  类型：回调函数
  可空：Y
  意思：输入数据接受回调函数地址
 参数.二：lParam
  In/Out：In/Out
  类型：无类型指针
  可空：Y
  意思：回调函数自定义参数

返回值

类型：逻辑型
意思：是否成功

备注

模式设置必须要回调函数有值才有作用
