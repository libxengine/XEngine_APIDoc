函数

NetCore_UDPXCore_GetFlowEx

说明

获取服务发送和接受的流量信息

语法

BOOL NetCore_UDPXCore_GetFlowEx(XHANDLE xhNet, __int64u * pInt_UPByte, __int64u * pInt_DWByte, __int64u * pInt_UPPkt = NULL, __int64u * pInt_DWPkt = NULL)

参数

 参数.一：pInt_UPByte
  In/Out：Out
  类型：四字指针
  可空：N
  意思：总共发送了多少字节
 参数.二：pInt_DWByte
  In/Out：Out
  类型：四字指针
  可空：N
  意思：总共接受了多少字节
 参数.三：pInt_UPPkt
  In/Out：Out
  类型：四字指针
  可空：Y
  意思：总共发送次数
 参数.四：pInt_DWPkt
  In/Out：Out
  类型：四字指针
  可空：Y
  意思：总共接受次数

返回值

类型：逻辑型
意思：是否获取成功

备注
