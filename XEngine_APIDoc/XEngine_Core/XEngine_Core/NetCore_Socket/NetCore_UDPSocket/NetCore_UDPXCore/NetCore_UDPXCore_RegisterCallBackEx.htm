函数

NetCore_UDPXCore_RegisterCallBackEx

说明

注册数据处理回调函数

语法

BOOL NetCore_UDPXCore_RegisterCallBackEx(XHANDLE xhNet,CALLBACK_NETCORE_SOCKET_NETEVENT_RECV fpCallePoll_Recv,LPVOID lRecv = NULL)

参数

  参数一：fpCallePoll_Recv
   In/Out：In/Out
   类型：回调函数指针
   可空：N
   意思：收到用户发送数据事件回调
  参数二：lRecv
   In/Out：In
   类型：无类型指针
   可空：Y
   意思：收到数据回调函数自定义参数

返回值

类型：逻辑型
意思：是否成功

备注

开启服务器成功后，立即调用此函数
