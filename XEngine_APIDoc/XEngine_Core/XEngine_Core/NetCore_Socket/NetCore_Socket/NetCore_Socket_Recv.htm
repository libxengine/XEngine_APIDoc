函数

NetCore_Socket_Recv

功能

从一套套接字上接受指定的大小数据

语法

XBOOL NetCore_Socket_Recv(XSOCKET hSocket, XCHAR* ptszRecvMsg, int* pInt_MsgLen)

参数

  参数一：hSocket
   In/Out：In
   类型：套接字句柄
   可空：N
   意思：要接受的套接字
  参数二：ptszRecvMsg
   In/Out：Out
   类型：字符指针
   可空：N
   意思：接受到的数据缓冲区
  参数三：pInt_MsgLen
   In/Out：In/Out
   类型：整数型指针
   可空：N
   意思：输入：表示接受数据缓冲区的大小，输出：实际接受数据的大小

返回值

类型：句柄
意思：是否成功

备注
