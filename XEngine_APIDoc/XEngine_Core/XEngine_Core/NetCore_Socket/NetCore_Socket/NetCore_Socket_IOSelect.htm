函数

NetCore_Socket_IOSelect

功能

选择模型核心函数封装

语法

BOOL NetCore_Socket_IOSelect(SOCKET hSocket, BOOL bRead = FALSE, int nTimeOut = 100

参数

 参数.一：hSocket
  In/Out：In
  类型：套接字句柄
  可空：N
  意思：要轮训的句柄
 参数.二：bRead
  In/Out：In
  类型：逻辑型
  可空：Y
  意思：读还是写
 参数.三：nTimeOut
  In/Out：In
  类型：整数型
  可空：Y
  意思：超时时间，如果为0表示没有超时时间，单位毫秒

返回值

类型：句柄
意思：是否成功

备注
