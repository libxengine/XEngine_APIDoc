函数

Protocol_UDPRaw_Packet

功能

组包一个UDP头

语法

BOOL Protocol_UDPRaw_Packet(CHAR * ptszMsgBuffer, int* pInt_Len, PROTOCOL_RAWSOCKET_NETPARAM * pSt_RAWSocket, LPCSTR lpszMsgBuffer = ("XEngine_Tracert"))

参数

 参数.一：ptszMsgBuffer
   In/Out：Out
   类型：字符指针
   可空：N
   意思：导出组好包的协议头数据
 参数.二：pInt_Len
   In/Out：Out
   类型：整数型指针
   可空：N
   意思：输出真实缓冲区大小
 参数.三：pSt_RAWSocket
   In/Out：In
   类型：数据结构指针
   可空：N
   意思：输入发送的协议参数
 参数.四：lpszMsgBuffer
   In/Out：In
   类型：常量字符指针
   可空：N
   意思：输入要打包的后续数据缓冲区

返回值

类型：逻辑型
意思：是否成功
