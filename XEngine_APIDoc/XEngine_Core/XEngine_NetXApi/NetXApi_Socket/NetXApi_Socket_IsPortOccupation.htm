函数

NetXApi_Socket_IsPortOccupation

功能

端口是否被使用

语法

BOOL NetXApi_Socket_IsPortOccupation(int nPort, int nProto)

参数

 参数.一：nPort
  In/Out：In
  类型：整数型
  可空：N
  意思：要检查的端口号码
 参数.二：nProto
  In/Out：In
  类型：整数型
  可空：Y
  意思：协议类型，要查找端口的所属协议

返回值

类型：逻辑型
意思：是否被使用，假为没有

备注

 

示例

if (NetXApi_Socket_IsPortOccupation(5001, XENGINE_NETXAPI_SOCKET_NETSTATE_PROTOCOL_TCP))
    {
        printf("占用\n");
    }
    else
    {
        printf("未占用\n");
    }
