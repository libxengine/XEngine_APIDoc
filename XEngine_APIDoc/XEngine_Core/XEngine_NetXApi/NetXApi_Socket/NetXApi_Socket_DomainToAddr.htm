函数

NetXApi_Socket_DomainToAddr

功能

名称转地址列表

语法

BOOL NetXApi_Socket_DomainToAddr(LPCSTR lpszDomain, CHAR * **pppszListAddr, int* pInt_ListCount)

参数

 参数.一：lpszDomain
  In/Out：In
  类型：常量字符指针
  可空：N
  意思：输入一个标准的名称
 参数.二：pppszListAddr
  In/Out：Out
  类型：三级指针
  可空：N
  意思：输出获取到的地址列表
 参数.三：pInt_ListCount
  In/Out：Out
  类型：整数型指针
  可空：N
  意思：输出列表个数

返回值

类型：逻辑型
意思：是否成功

备注

 

示例

    nListCount = 0;
    CHAR** ppszListAddr;
    NetXApi_Socket_DomainToAddr("www.baidu.com", &ppszListAddr, &nListCount);
    for (int i = 0; i < nListCount; i++)
    {
        printf("NetXApi_Socket_DomainToAddr:%s\n", ppszListAddr[i]);
    }
    BaseLib_OperatorMemory_Free((XPPPMEM)&ppszListAddr, nListCount);
