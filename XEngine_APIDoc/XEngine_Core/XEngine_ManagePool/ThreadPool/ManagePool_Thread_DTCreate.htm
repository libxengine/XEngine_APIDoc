函数

ManagePool_Thread_DTCreate

功能

创建线程池

语法

XHANDLE ManagePool_Thread_DTCreate(int nThreadCount,int nMaxTask = 10000,BOOL bIsClear = TRUE)

参数

 参数.一：nThreadCount
  In/Out：In
  类型：整数型
  可空：N
  意思：可运行线程池数量
 参数.二：nMaxTask
  In/Out：In
  类型：整数型
  可空：Y
  意思：每个线程最多可运行执行多少任务，默认10000
 参数.三：bIsClear
  In/Out：In
  类型：逻辑型
  可空：Y
  意思：超过指定任务大小是否清理任务池，默认为真

返回值

类型：句柄型
意思：成功返回创建的句柄,失败返回NULL
