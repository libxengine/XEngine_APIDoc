函数

ManagePool_Thread_NQPriority

功能

设置线程优先级

语法

bool ManagePool_Thread_NQPriority(XHANDLE xhPool, int nThreadNumber, ENUM_XENGINE_MANAGEPOOL_THREAD_PRIORITY enThreadPriority)

参数

 参数.一：xhPool
  In/Out：In
  类型：句柄
  可空：N
  意思：输入要操作的线程池
 参数.二：nThreadNumber
  In/Out：In
  类型：整数型
  可空：N
  意思：输入要操作的线程编号,编号从0开始,比如4个线程,编号就是0,1,2,3
 参数.三：enThreadPriority
  In/Out：In
  类型：枚举型
  可空：N
  意思：输入要设置的优先级

返回值

类型：逻辑型
意思：是否成功

备注

某些系统需要管理员权限,某些系统可能不支持设置,比如IOS
