函数

WBlackList_Domain_GetList

功能

获取当前黑名单地址列表

语法

BOOL WBlackList_Domain_GetList(XNETHANDLE xhNet, CHAR * **pppszListBlack = NULL, int* pInt_BlackCount = NULL, CHAR * **pppszListWhite = NULL, int* pInt_WhiteCount = NULL)

参数

 参数.一：xhNet
  In/Out：In
  类型：句柄
  可空：N
  意思：要操作的句柄
 参数.二：pppszListBlack
  In/Out：Out
  类型：三级指针
  可空：Y
  意思：导出获取到的黑名单地址列表
 参数.三：pInt_BlackCount
  In/Out：Out
  类型：整数型指针
  可空：Y
  意思：导出获取到的黑名单地址个数
 参数.四：pppszListWhite
  In/Out：Out
  类型：三级指针
  可空：Y
  意思：导出获取到的白名单地址列表
 参数.五：pInt_WhiteCount
  In/Out：Out
  类型：整数型指针
  可空：Y
  意思：导出获取到的白名单地址个数

返回值

类型：逻辑型
意思：是否成功
