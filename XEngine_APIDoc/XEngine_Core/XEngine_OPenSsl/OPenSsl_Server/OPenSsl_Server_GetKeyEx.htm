函数

OPenSsl_Server_GetKeyEx

功能

获取指定加密类型的密钥信息

语法

bool OPenSsl_Server_GetKeyEx(XHANDLE xhToken, LPCXSTR lpszClientAddr, XBYTE* ptszKEYBuffer, LPCXSTR lpszKeyTStr = _X("EXTRACTOR-dtls_srtp"), int nKeySize = 30)

参数

 参数.一：lpszClientAddr
  In/Out：Out
  类型：字符指针
  可空：N
  意思：输出本地发送KEY数据密钥
 参数.二：ptszKEYBuffer
  In/Out：Out
  类型：字符指针
  可空：N
  意思：导出密钥缓冲区
 参数.三：lpszKeyTStr
  In/Out：Out
  类型：常量字符指针
  可空：Y
  意思：输入密钥获取类型字符串
 参数.四：nKeySize
  In/Out：In
  类型：整数型
  可空：Y
  意思：根据密钥不同加密算法不同大小也不一样,DTSL-AES128的大小是30字节

返回值

类型：逻辑型
意思：是否成功

备注
