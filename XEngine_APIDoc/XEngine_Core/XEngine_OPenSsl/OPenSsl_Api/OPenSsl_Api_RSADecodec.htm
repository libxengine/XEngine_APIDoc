函数

OPenSsl_Api_RSADecodec

功能

使用RSAKEY解密数据

语法

BOOL OPenSsl_Api_RSADecodec(LPCTSTR lpszKeyFile, const UCHAR* lpszSourceData, int* pInt_Len, TCHAR* ptszDestData, BOOL bKeyType = TRUE, LPCTSTR lpszPriPass = NULL)

参数

 参数.一：lpszKeyFile
  In/Out：In
  类型：常量字符指针
  可空：N
  意思：KEY文件路径
 参数.二：lpszSourceData
  In/Out：In
  类型：常量无符号字符指针
  可空：N
  意思：要解密的原始数据
 参数.三：pInt_Len
  In/Out：In/Out
  类型：整数型指针
  可空：N
  意思：输入：原始数据长度，输出解密后的数据长度
 参数.四：ptszDestData
  In/Out：Out
  类型：字符指针
  可空：N
  意思：输出解密后的数据
 参数.五：bKeyType
  In/Out：In
  类型：逻辑型
  可空：Y
  意思：为真表示使用公钥解密，为假表示使用私钥解密
 参数.六：lpszPriPass
  In/Out：In
  类型：常量字符指针
  可空：Y
  意思：如果私钥加过密,那么必须输入私钥密码

返回值

类型：逻辑型
意思：是否解密成功
