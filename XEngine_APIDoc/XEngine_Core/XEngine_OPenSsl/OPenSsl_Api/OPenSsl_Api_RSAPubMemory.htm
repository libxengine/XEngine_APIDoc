函数

OPenSsl_Api_RSAPubMemory

功能

公钥数据生成到内存函数

语法

bool OPenSsl_Api_RSAPubMemory(XCHAR* ptszPubBuffer, int* pInt_PLen, LPCXSTR lpszPriBuffer, int nPLen, LPCXSTR lpszPassStr = NULL)

参数

 参数.一：ptszPubBuffer
  In/Out：Out
  类型：字符指针
  可空：N
  意思：输出的公钥数据
 参数.二：pInt_PLen
  In/Out：In
  类型：整数型指针
  可空：N
  意思：输出的公钥数据大小
 参数.三：lpszPriBuffer
  In/Out：In
  类型：常量字符指针
  可空：N
  意思：输入私钥内存缓冲区
 参数.四：nPLen
  In/Out：In
  类型：整数型
  可空：N
  意思：输入私钥内存大小
 参数.五：lpszPassStr
  In/Out：In
  类型：常量字符指针
  可空：Y
  意思：私钥的密码

返回值

类型：逻辑型
意思：是否生成成功
