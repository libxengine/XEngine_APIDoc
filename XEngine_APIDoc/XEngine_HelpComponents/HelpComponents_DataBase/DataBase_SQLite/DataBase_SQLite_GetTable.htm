函数

DataBase_SQLite_GetTable

功能

获取查询语句返回的值

语法

BOOL DataBase_SQLite_GetTable(XHDATA xhData,LPCSTR lpszSQL,char ***ppTszString,int *pInt_Row,int *pInt_Column)

参数

 参数.一：xhData
  In/Out：In
  类型：数据库句柄
  可空：N
  意思：要对哪个数据库进行操作
 参数.二：lpszSQL
  In/Out：In
  类型：常量字符指针
  可空：N
  意思：要执行的SQL语句
 参数.三：ppTszString
  In/Out：Out
  类型：指向字符指针的指针的指针
  可空：N
  意思：保存结果的记录集数据  ppTszString[i]
 参数.四：pInt_Row
  In/Out：Out
  类型：整数型指针
  可空：N
  意思：执行后影响的行数
 参数.五：pInt_Column
  In/Out：Out
  类型：整数型指针
  可空：N
  意思：执行后影响的列数

返回值

类型：逻辑型
意思：是否成功
