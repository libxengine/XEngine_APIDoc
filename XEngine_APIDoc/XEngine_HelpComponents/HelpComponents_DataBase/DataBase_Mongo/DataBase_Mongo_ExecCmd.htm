函数

DataBase_Mongo_ExecCmd

功能

执行一条命令并且获得回复

语法

BOOL DataBase_Mongo_ExecCmd(XNETHANDLE xhNet,LPCSTR lpszDBName,LPCSTR lpszBsonDoc,CHAR *ptszJsonBuffer,int *pInt_JsonLen,BOOL bBson = TRUE)

参数

 参数.一：xhNet
  In/Out：In
  类型：句柄
  可空：N
  意思：输入数据库句柄
 参数.二：lpszDBName
  In/Out：In
  类型：常量字符指针
  可空：N
  意思：要操作的数据库名
 参数.三：lpszBsonDoc
  In/Out：In
  类型：常量字符指针
  可空：N
  意思：要操作的BSON结构
 参数.四：ptszJsonBuffer
  In/Out：Out
  类型：常量字符指针
  可空：N
  意思：导出服务器的返回内容，JSON格式。
 参数.五：pInt_JsonLen
  In/Out：In/Out
  类型：整数型指针
  可空：N
  意思：输入提供的缓冲区大小，输出获取到的缓冲区大小
 参数.六：bBson
  In/Out：In
  类型：逻辑型
  可空：Y
  意思：输入数据是BSON还是JSON格式，默认BSON

返回值

类型：逻辑型
意思：是否成功
