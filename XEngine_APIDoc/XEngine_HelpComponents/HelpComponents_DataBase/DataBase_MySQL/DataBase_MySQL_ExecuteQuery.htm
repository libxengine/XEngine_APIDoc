函数

DataBase_MySQL_ExecuteQuery

功能

执行查询语句

语法

BOOL DataBase_MySQL_ExecuteQuery(XHDATA xhData,XHDATA *pxhResult,LPCSTR lpszSQLQuery,__int64u *pullLine,__int64u *pullField)

参数

 参数.一：xhData
  In/Out：In
  类型：数据库句柄
  可空：N
  意思：要对哪个SQL客户端执行
 参数.二：pxhResult
  In/Out：Out
  类型：记录集句柄
  可空：N
  意思：导出查询的数据库记录集
 参数.三：lpszSQLQuery
  In/Out：In
  类型：常量字符指针
  可空：N
  意思：SQL语句
 参数.四：pullLine
  In/Out：Out
  类型：双字64位
  可空：N
  意思：结果集的行数
 参数.五：pullField
  In/Out：Out
  类型：双字64位
  可空：N
  意思：结果集的列数

返回值

类型：逻辑型
意思：是否成功
