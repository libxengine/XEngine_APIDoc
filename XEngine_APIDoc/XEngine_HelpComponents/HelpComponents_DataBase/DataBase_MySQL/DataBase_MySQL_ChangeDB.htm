函数

DataBase_MySQL_ChangeDB

功能

以当前身份改变数据库

语法

BOOL DataBase_MySQL_ChangeDB(XHDATA xhData,DATABASE_MYSQL_CONNECTINFO st_MySQLConnector)

参数

 参数.一：xhData
  In/Out：In
  类型：数据库句柄
  可空：N
  意思：要对哪个SQL客户端执行
 参数.二：st_MySQLConnector
  In/Out：In
  类型：结构体
  可空：N
  意思：用户信息结构

返回值

类型：逻辑型
意思：是否成功

备注

第二个参数只有DBNAME有效，其他成员必须与之前保持一致
