函数

DataBase_MySQL_Execute

功能

执行非查询语句

语法

bool DataBase_MySQL_Execute(XHDATA xhData,LPCSTR lpszSQLQuery, __int64u *pInt_Rows = NULL)

参数

 参数.一：xhData
  In/Out：In
  类型：数据库句柄
  可空：N
  意思：要对哪个SQL客户端执行
 参数.二：lpszSQLQuery
  In/Out：In
  类型：常量字符指针
  可空：N
  意思：要执行的SQL语句
 参数.三：pInt_Rows
  In/Out：In/Out
  类型：整数型指针
  可空：Y
  意思：输入数据大小,输出受影响的行数

返回值

类型：逻辑型
意思：是否成功
