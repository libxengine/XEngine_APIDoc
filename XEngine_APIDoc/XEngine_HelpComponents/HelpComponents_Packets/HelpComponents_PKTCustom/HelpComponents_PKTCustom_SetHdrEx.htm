函数

HelpComponents_PKTCustom_SetHdrEx

功能

设置协议头大小和后续字段位置

语法

BOOL HelpComponents_PKTCustom_SetHdrEx(XHANDLE xhToken, int nPosStart, int nPosEnd, int nHdrSize = 0, BOOL bEndian = TRUE)

参数

 参数.一：nHdrSize
   In/Out：In
   类型：整数型
   可空：N
   意思：协议头大小
 参数.二：nPosStart
   In/Out：In
   类型：整数型
   可空：N
   意思：协议头后续数据大小起始位置
 参数.三：nPosEnd
   In/Out：In
   类型：整数型
   可空：Y
   意思：协议头后续数据大小结束位置
 参数.四：bEndian
   In/Out：In
   类型：逻辑型
   可空：Y
   意思：表示大小的字段是否需要转换字节序(ntohs),默认是

返回值

类型：逻辑型
意思：是否成功

备注

此函数必须调用,如果是固定大小,只需要调用此函数即可
