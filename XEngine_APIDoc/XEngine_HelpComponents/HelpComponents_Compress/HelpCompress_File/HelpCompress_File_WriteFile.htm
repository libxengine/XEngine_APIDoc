函数

HelpCompress_File_WriteFile

功能

吧当前句柄位置的压缩文件数据写到指定位置

语法

BOOL HelpCompress_File_WriteFile(XZIP pxZip, LPCSTR lpszFile)

参数

 参数.一：pxZip
  In/Out：In
  类型：ZIP句柄
  可空：N
  意思：打开的ZIP文件的句柄
 参数.二：lpszFile
  In/Out：In
  类型：常量字符指针
  可空：N
  意思：要写到哪个地方

返回值

类型：逻辑型
意思：是否成功

备注

需要使用上面的跳转函数定位好后的文件才可以使用此函数
