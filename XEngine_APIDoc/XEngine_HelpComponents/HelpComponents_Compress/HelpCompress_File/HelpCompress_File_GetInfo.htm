函数

HelpCompress_File_GetInfo

功能

获取当前文件信息

语法

BOOL HelpCompress_File_GetInfo(XZIP pxZip, HELPCOMPRESS_ZLIBINFO *pSt_FileInfo)

参数

 参数.一：pxZip
  In/Out：In
  类型：ZIP句柄
  可空：N
  意思：打开的ZIP文件的句柄
 参数.二：pSt_FileInfo
  In/Out：Out
  类型：结构体指针
  可空：N
  意思：获取到的文件信息

返回值

类型：逻辑型
意思：是否成功

备注

需要使用上面的跳转函数定位好后的文件才可以使用此函数
