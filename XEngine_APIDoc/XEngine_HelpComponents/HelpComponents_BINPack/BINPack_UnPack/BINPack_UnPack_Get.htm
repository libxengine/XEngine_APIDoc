函数

BINPack_UnPack_Get

功能

从一个打包的二进制数据中获取一段数据

语法

BOOL BINPack_UnPack_Get(XNETHANDLE xhFile, LPCSTR lpszFileName = NULL, CHAR *pszMemBuffer = NULL, int *pIntMemLen = NULL)

参数

 参数.一：xhFile
  In/Out：In
  类型：文件句柄
  可空：N
  意思：输入你刚才初始化的网络句柄
 参数.二：lpszFileName
  In/Out：In
  类型：常量字符指针
  可空：Y
  意思：获取到的数据是否写成文件,如果为NULL,那么下一个参数不能为NULL
 参数.三：pszMemBuffer
  In/Out：In
  类型：字符指针
  可空：Y
  意思：获取到的文件导出到内存
 参数.四：pIntMemLen
  In/Out：In
  类型：整数型指针
  可空：Y
  意思：输入你提供的大小,输出真实大小,不够将返回错误码

返回值

类型：逻辑型
意思：是否成功
