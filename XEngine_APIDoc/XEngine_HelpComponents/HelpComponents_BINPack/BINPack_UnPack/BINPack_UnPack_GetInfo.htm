函数

BINPack_UnPack_GetInfo

功能

获取信息

语法

BOOL BINPack_UnPack_GetInfo(XNETHANDLE xhFile, __int64x* pInt_AllSize = NULL, __int64x* pInt_FSize = NULL, int* pInt_Count = NULL, int* pInt_Index = NULL)

参数

 参数.一：xhFile
  In/Out：In
  类型：文件句柄
  可空：N
  意思：要操作的文件句柄
 参数.二：pInt_AllSize
  In/Out：Out
  类型：整数型指针
  可空：Y
  意思：输出要解包文件的总大小
 参数.三：pInt_FSize
  In/Out：Out
  类型：整数型指针
  可空：Y
  意思：输出只是文件的总大小
 参数.四：pInt_Count
  In/Out：Out
  类型：整数型指针
  可空：Y
  意思：输出文件个数
 参数.五：pInt_Index
  In/Out：Out
  类型：整数型指针
  可空：Y
  意思：输出当前索引

返回值

类型：逻辑型
意思：是否成功
