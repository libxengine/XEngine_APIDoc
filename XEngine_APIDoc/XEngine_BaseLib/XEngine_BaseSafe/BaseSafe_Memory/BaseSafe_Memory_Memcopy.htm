函数

BaseSafe_String_Memcopy

功能

安全的内存拷贝函数

语法

bool BaseSafe_Memory_Memcopy(XBYTE* ptszDestByte, size_t nDestSize, LPCXBTR lpszSourceByte, size_t nSourceSize)

参数

 参数.一：ptszDestByte
  In/Out：Out
  类型：字符指针
  可空：N
  意思：输出拷贝的内存
 参数.三：nDestSize
  In/Out：In
  类型：整数型
  可空：N
  意思：输入目标缓冲区最大大小
 参数.二：lpszSourceByte
  In/Out：In
  类型：常量字符指针
  可空：N
  意思：输入要拷贝的内存
 参数.三：nDestSize
  In/Out：In
  类型：整数型
  可空：N
  意思：输入要拷贝的缓冲区最大大小

返回值

类型：整数型
意思：拷贝大小

备注

如果他们刚好一样大,那么会丢失最后一个字符,因为最后一个字符会被设置为字符串结束
