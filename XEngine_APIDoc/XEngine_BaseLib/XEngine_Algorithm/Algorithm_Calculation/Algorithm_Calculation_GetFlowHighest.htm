函数

Algorithm_Calculation_GetFlowHighest

功能

获取发送接受流量以来每秒最高平均速度

语法

bool Algorithm_Calculation_GetFlowHighest(XHANDLE pxhToken, __int64u* pInt_SDFlow = NULL, __int64u* pInt_RVFlow = NULL)

参数

 参数.一：pxhToken
  In/Out：In
  类型：句柄
  可空：N
  意思：输入要操作的句柄
 参数.二：pInt_SDFlow
  In/Out：Out
  类型：整数型指针
  可空：N
  意思：输出发送的最高每秒平均流量
 参数.三：pInt_RVFlow
  In/Out：In
  类型：整数型指针
  可空：Y
  意思：输出接受的最高每秒平均流量

返回值

类型：逻辑型
意思：是否成功

备注

需要启用时间追踪才有效
