函数

Algorithm_Calculation_Reset

功能

重置数据

语法

BOOL Algorithm_Calculation_Reset(XHANDLE pxhToken, int nTraceTime = 0, BOOL bTime = TRUE, BOOL bSDFlow = TRUE, BOOL bRVFlow = TRUE)

参数

 参数.一：pxhToken
  In/Out：In
  类型：句柄
  可空：N
  意思：输入要操作的句柄
 参数.二：nTraceTime
  In/Out：In
  类型：整数型
  可空：Y
  意思：追溯时间.为0不修改
 参数.三：bTime
  In/Out：In
  类型：逻辑型
  可空：Y
  意思：是否重置次数信息
 参数.四：bSDFlow
  In/Out：In
  类型：逻辑型
  可空：Y
  意思：是否重置发送流量
 参数.五：bRVFlow
  In/Out：In
  类型：逻辑型
  可空：Y
  意思：是否重置接受流量

返回值

类型：逻辑型
意思：是否成功
