函数

BaseLib_String_DelSub

功能

从一个指定的字符串中删除指定字符串

语法

bool BaseLib_String_DelSub(XCHAR* ptszSource, LPCXSTR lpszDelString, bool bDelAll = true)

参数

 参数.一：ptszSource
  In/Out：In/Out
  类型：字符指针
  可空：N
  意思：输入：待删除的字符串缓冲区，导出，删除后的字符串
 参数.二：lpszDelString
  In/Out：In
  类型：常量字符指针
  可空：N
  意思：要删除的字符串
 参数.三：bDelAll
  In/Out：In
  类型：逻辑型
  可空：Y
  意思：是否删除所有指定的字符串

返回值

类型：逻辑型
意思：是否成功
