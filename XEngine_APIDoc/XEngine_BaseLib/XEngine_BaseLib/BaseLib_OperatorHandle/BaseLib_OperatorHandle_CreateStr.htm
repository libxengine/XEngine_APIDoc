函数

BaseLib_OperatorHandle_CreateStr

功能

创建指定位数随机字符串

语法

BOOL BaseLib_OperatorHandle_CreateStr(CHAR* ptszKey, int nSize = 16, int nType = 0, int nCharType = 0)

参数

 参数.一：ptszKey
  In/Out：Out
  类型：字符指针
  可空：N
  意思：输出创建后的值
 参数.二：nSize
  In/Out：In
  类型：整数型
  可空：Y
  意思：输入要创建多少位
 参数.三：nType
  In/Out：In
  类型：整数型
  可空：Y
  意思：创建类型,默认0随机,1为全数字,2为全字母
 参数.四：nCharType
  In/Out：In
  类型：整数型
  可空：Y
  意思：字母类型,0为随机,1全部大写,2全部小写

返回值

类型：逻辑型
意思：是否创建成功
