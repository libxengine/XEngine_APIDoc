函数

BaseLib_TimeTigger_Set

功能

设置添加一个触发器

语法

bool BaseLib_TimeTigger_Set(XHANDLE pxhTimer, int nIDEvent, __int64x nMillTimer = 0, int nCount = 1)

参数

 参数.一：pxhTimer
  In/Out：In
  类型：触发器句柄
  可空：N
  意思：输入要操作的触发器
 参数.二：nIDEvent
  In/Out：In
  类型：整数型
  可空：N
  意思：输入触发器ID.这个ID不能重复
 参数.三：nMillTimer
  In/Out：In
  类型：整数型
  可空：Y
  意思：输入间隔多久触发一次,单位纳秒
 参数.四：nCount
  In/Out：In
  类型：整数型
  可空：Y
  意思：输入触发器触发次数.为-1表示一直触发,为0不触发

返回值

类型：逻辑型
意思：是否成功
