函数

BaseLib_IO_SetBufferSize

功能

设置套接字缓冲区大小

语法

bool BaseLib_IO_SetBufferSize(XSOCKET hSocket, int nSDMax = 0, int nRVMax = 0)

参数

 参数.一：hSocket
  In/Out：In
  类型：套接字句柄
  可空：N
  意思：输入要操作的套接字
 参数.二：nSDMax
  In/Out：In
  类型：整数型
  可空：N
  意思：发送缓冲区大小
 参数.三：nRVMax
  In/Out：In
  类型：整数型
  可空：N
  意思：接受缓冲区大小

返回值

  类型：逻辑型
  意思：是否成功
