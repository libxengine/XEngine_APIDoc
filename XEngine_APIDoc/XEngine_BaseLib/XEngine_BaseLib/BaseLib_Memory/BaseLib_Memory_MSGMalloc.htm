函数

BaseLib_Memory_MSGMalloc

功能

申请MSG结构数组内存

语法

bool BaseLib_Memory_MSGMalloc(XENGINE_MSGBUFFER* pSt_MSGBuffer)

参数

 参数.一：pSt_MSGBuffer
  In/Out：In
  类型：数据结构指针
  可空：N
  意思：要初始化使用的内存数据结构

返回值

类型：逻辑型
意思：是否成功

备注

pSt_MSGBuffer->nMSGLen[]需要填充好每个元素的大小,系统会自动初始化并且申请内存,不使用的元素大小设置为0即可
申请的内存需要使用BaseLib_Memory_MSGFree函数进行释放
