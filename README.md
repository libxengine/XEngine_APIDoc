# XEngine_OPenSource

本仓库是XEngine的API接口文档地址  
本文档系统使用winchm生成  

## 介绍

基于HTML页面生成的接口文档系统.暂时了系统支持的所有API函数以及其解释  
当前支持的XEngine API版本:V8.24


## 安装教程

你需要安装winchm程序后才能编译和使用此套API系统.
